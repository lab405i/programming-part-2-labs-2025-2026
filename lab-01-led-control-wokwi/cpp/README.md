# C++ Implementation — LED Control in Wokwi

## Description

This folder contains the **C++ implementation** for **Laboratory Work 1: LED Control in Wokwi Using C, C++ and MicroPython**.

The purpose of this part is to demonstrate GPIO control of the Raspberry Pi Pico using the C++ programming language and an object-oriented programming approach.

The program controls an LED connected to GPIO pin **GP15** and makes it blink with a fixed delay.

The same Wokwi circuit is used for all three implementations: C, C++ and MicroPython.
