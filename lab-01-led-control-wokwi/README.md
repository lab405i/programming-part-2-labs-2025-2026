# Laboratory Work 1  
# LED Control in Wokwi Using C, C++ and Python

## Topic

Investigation of embedded systems programming using C, C++ and Python in the Wokwi simulator based on LED control.

## Course

**Programming. Part 2**  
Kharkiv National University of Radio Electronics  
2025–2026

## Goal

The goal of this laboratory work is to study the basic principles of embedded systems programming using different programming languages and approaches.

During the laboratory work, students implement the same embedded system task — LED control — using three programming languages:

- C;
- C++;
- Python / MicroPython.

# The same Wokwi circuit is used for all three implementations: C, C++ and MicroPython.

See diagram.json

The work is performed in the **Wokwi online simulator** using the **Raspberry Pi Pico** microcontroller.
