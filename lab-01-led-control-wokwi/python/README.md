# MicroPython Implementation — LED Control in Wokwi

## Description

This folder contains the **MicroPython implementation** for **Laboratory Work 1: LED Control in Wokwi Using C, C++ and MicroPython**.

The purpose of this part is to demonstrate basic GPIO control of the Raspberry Pi Pico using MicroPython.  
The program controls an LED connected to GPIO pin **GP15** and makes it blink with a fixed delay.
