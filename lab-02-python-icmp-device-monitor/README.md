## Description

This laboratory work is devoted to the development of a simple network monitoring tool using Python.

The program checks the availability of a network device by its **IP address** or **domain name**.  
If the device is available, the program displays ping statistics.  
If the device is unavailable, the program simulates a reboot process, plays sound notifications, displays a progress bar and shows Windows message boxes.

This laboratory work demonstrates the practical use of Python for network diagnostics, command-line arguments, ICMP requests, sound notifications and simple graphical system messages.

## Goal

The goal of this laboratory work is to study the basic principles of network device monitoring using Python and ICMP requests.

Students will learn how to:

```text
use command-line arguments in Python;
check the availability of a network device;
resolve a domain name into an IP address;
send ICMP ping requests;
process ping statistics;
simulate a device reboot process;
display progress in the console;
use Windows sound notifications;
use simple graphical message boxes.
